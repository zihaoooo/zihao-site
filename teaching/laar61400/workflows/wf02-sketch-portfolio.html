Site Reading Deck
01 Setup — Scan and Gather
Scan your sketches. Collect site photographs, satellite imagery, and historical maps. Organize into folders before opening Photoshop.
  • Scan at 300 dpi, grayscale. Adobe Scan or Microsoft Lens work fine for phone capture.
  • Photograph at three scales: wide, medium, and close-up material shots
  • Satellite and historical maps: download at the scale you intend to use
02 Build the Site Joiner Photoshop
Assemble overlapping site photos into a composite — reference David Hockney's joiners. Multiple perspectives, visible seams. Use T (Transform) to scale and position each piece.
  • Don't stitch seamlessly — the overlap and fragmentation are part of the drawing
  • Try a horizontal strip for a spatial sequence; a grid for material close-ups
  • Expand the canvas as you go: Image → Canvas Size
03 Cut, Layer, and Blend Photoshop
Use the Object Selection tool W to remove backgrounds and isolate elements. Cut up and rearrange images freely. Layer maps and satellite imagery using blending modes so both read simultaneously.
  • Multiply — darkens; works for linework maps over photos
  • Screen — lightens; works for satellite imagery over darker layers
  • Adjust layer opacity (30–60%) to control how much each layer shows through
04 Draw and Annotate Photoshop
Print a portion of your composition. Place trace paper over it and draw — trace edges, hatch surfaces, annotate conditions. Rescan and place back as a layer set to Multiply — the paper disappears, only the marks remain. Add text at variable scale over the image (reference Julie Bargmann's site collages).
  • Trace paper lets you draw over the print without committing — pull it off, adjust, redraw
  • Scanned drawing layer → blending mode: Multiply
  • Text is part of the drawing, not a caption — vary scale and weight deliberately
  • Subjects: site conditions, material notes, measurements, observations, questions
05 Set Up the InDesign Deck InDesign
New multi-page document: 11"×17" landscape, 0.125" bleed. Build a parent page with repeating elements. Define paragraph styles for title, caption, and annotation — apply consistently, never format manually.
  • Plan pages before building: joiner spread · texture palette · additional analytical pages
  • Parent page: page number, course name, margin rules
  • Paragraph Styles F11: Title, Caption, Annotation — at minimum
06 Compose the Pages InDesign
Joiner spread. Place the Photoshop export — full bleed or within margins. Float analytical text frames over the image. The page should feel like a drawing, not a photograph layout.
  • Ctrl D to place; Ctrl Alt Shift E to fill frame
  • Keep text on its own layer above the image
  • Direct Selection tool A to reposition image within crop
Texture palette. A small-multiple grid of material close-ups — equal frames, each captioned with material, location, and a brief condition note. A site texture inventory.
  • Step and Repeat Ctrl Alt U to build the grid
  • Select all frames → Ctrl D to load and place multiple images at once
  • Caption format: material · location · one observation
07 Export PDF InDesign
Export the full deck as a single PDF.
  • Ctrl E → PDF (Print) with bleed and crop marks for physical output
  • PDF (Interactive) for screen sharing — smaller file, no bleed needed
  • Package Ctrl Alt Shift P before sharing to collect all linked files
