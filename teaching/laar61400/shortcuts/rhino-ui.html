Rh Rhino 3D — Navigation & UI

Interface orientation for new users
Mouse Controls
Mouse Controls
Left click — Select object
Left drag — Window select (L→R solid, R→L crossing)
Right click — Confirm / repeat last command
Right drag — Rotate view (Perspective only)
Middle drag — Pan view (all viewports)
Scroll wheel — Zoom in / out
Shift + Right drag — Pan (alternate)
Ctrl + Right drag — Rotate (alternate)
Viewport Layout
Viewport Layout
Double-click any viewport title to maximize it. Double-click again to restore 4-view.
Right-click the viewport title to change display mode (Wireframe, Shaded, Rendered…).
Top / Front / Right are parallel projections — no perspective distortion. Use for precise drawing.
Perspective is the only viewport where Right-drag rotates the view.
Named views: type NamedView to save and restore camera positions.
Status Bar Toggles — click to turn on/off, right-click to configure
Status Bar Toggles

Display Modes — right-click viewport title to switch
Display Modes
WireframeEdges only. Fastest. Use for modeling.
ShadedSolid surfaces with basic lighting.
RenderedMaterials and lights visible. Slower.
GhostedTransparent surfaces. See through objects.
X-RayAll edges visible through surfaces.

Coordinate Input — type directly at the command prompt
Coordinate Input
Absolute: type x,y,z e.g. 0,0,0  ·  Relative: type r x,y,z e.g. r 5,0,0 (5 units in X from last point)  ·  Distance + direction: move cursor in direction, type a number and press Enter — Rhino draws that exact distance along the cursor's direction.  ·  Angle constraint: type < 45 to constrain to 45°.
Rhino 3D — Windows version. Mac controls are largely identical; substitute Cmd for Ctrl where applicable.
